Fly to the Limit is a ficticous plane company that offers tours in the sky on a weekly basis.

For this website project, I ended up using a JQuery Lightbox plugin found on Magnific Pop-up and a date picker plugin from the Rome Date Picker to add more interactivity to the site and allow users to apply for tours online from the website.

Extra interactive features I added into the website include CSS animation transitions, a parallax banner image, basic Javascript form validation development and autoscroll features in JQuery to give the user of the site a better experience.
